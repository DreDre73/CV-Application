# CV/Resume Builder React App

A responsive web application to create, view, and download a professional CV/resume. Built with **React**, this project allows users to input personal, education, and work experience information and generate a styled PDF version of their resume.

---

## Features

- **Interactive Forms:** Input sections for personal info, education, work experience, skills, and projects.
- **Dynamic PDF Preview:** See your CV update in real time as you edit information.
- **Downloadable PDF:** Export your resume with a single click.
- **Custom Styling:** Predefined themes and layouts for a professional appearance.

---

## Tech Stack

- **Frontend:** React, HTML, CSS
- **PDF Generation:** `react-pdf`
- **Deployment:** Netlify

---

## Usage

1. Fill out your personal information, education, work experience, and projects.
2. Preview your CV in real-time in the PDF viewer.
3. Click **Download PDF** to export your resume.
